Stellar Program 又名恒星计划是 Solar Network 的会员系统，
参与其的成员可以获得一些特殊的福利和权限以访问新功能。

## 等级

Stellar Program 订阅分为三级，分别是 Stellar, Nova 和 Supernova。
其对应的翻译为「恒星」、「新星」与「超新星」。*亦或是「恒星」、「恒星+」、「恒星++」*

每一等级都包含上一等级的全部功能，并有相应的扩充。

## Perks

1. 等级「恒星」
    - 炫酷彩色用户名
    - 抢先体验新功能
    - 1.5x 经验获得增速
    - 增加领域、机器人、发布者额度
2. 等级「新星」
    - 等级「恒星」的所有功能
    - 优先功能请求
    - 2x 经验获取增速
    - 发布者「关注需批准」与「发布内容需关注后可见」功能
    - 增加更多领域、机器人、发布者额度
3. 等级「超新星」
    - 等级「新星」的所有功能
    - 2.5x 经验获取增速
    - 优先客户服务支持
    - 源码级技术支持
    - 增加超多领域、机器人、发布者额度

## 获得

用户有三种方式获得 Stellar Program，**在应用内通过源点购买**、**通过赞助支持 Solar Network 的开发时的赠品**，或者**通过兑换其他用户的礼物订阅**。

!!! warning

    现在不再可以用过应用内**源点**购买的方式获得 Stellar Program Supernova，包括赠礼与自用。

### 应用内购买

应用内通过 Solarpay 来购买需要您提前设置好 PIN Code 以便支付订单。详情请转到「源点钱包」章节。

在应用内购买的价格为：

- 等级「恒星」：每月 1200 源点，需要用户等级达到 30 级
- 等级「新星」：每月 2400 源点，需要用户等级达到 60 级
- ~~等级「超新星」：每月 3600 源点，需要用户等级达到 90 级~~

### 赞助开发时的赠品

在通过赞助获得 Stellar Program 之前，
您需要转到帐号设置中连接您要赞助要通过的平台的帐号，以便自动获取订单状态。

尽管在回复购买的时候需要提供订单号，但是您仍需要链接对应的帐号来获得 Stellar Program 订阅。

## 礼物订阅

礼物订阅允许你购买恒星计划订阅作为礼物送给其他用户（或作为公开礼物供任何人领取）。

### 主要功能

- **礼物码：** 每份礼物都有唯一的兑换码
- **灵活兑换：** 公开礼物可被任何人领取；定向礼物仅限指定接收者
- **自动清理：** 未支付的礼物会被隐藏并在 24 小时后自动移除

### 发送礼物

购买礼物订阅时，系统会生成一个唯一的礼物码。你可以将此码分享给目标接收者（或公开发布）。

### 兑换礼物

要兑换礼物：

1. 前往礼物兑换页面
2. 输入礼物码
3. 如果礼物有效且适用于你的账户，订阅将被激活

!!! note

    礼物码只能兑换一次。如果接收者已有活跃的订阅，礼物订阅将会**排队**，在当前订阅到期后自动激活。

### 注意事项

- 礼物订阅遵循与普通订阅相同的等级体系
- 接收者获得所赠等级的全部权益
- 如果接收者已有活跃订阅，礼物订阅会排队等待到期后激活
- 未支付的礼物在 24 小时后过期并自动清理
- 已兑换的礼物不可转让或退款
