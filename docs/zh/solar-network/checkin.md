# 签到系统

每日签到可以获得奖励！

## 如何签到

1. 进入 **仪表盘**
2. 点击签到按钮

## 签到奖励

| 奖励 | 说明 |
|------|------|
| 经验值 | 固定 100 点 |
| 积分 | 固定 10 积分 |

## 运势结果

签到时会随机获得一种运势：

| 运势 | 概率 |
|------|------|
| 大凶 | 10% |
| 凶 | 20% |
| 中平 | 40% |
| 吉 | 20% |
| 大吉 | 10% |

!!! note

    如果您在生日当天签到，将获得「生日快乐」特别运势！

## 签到小贴士

签到后会获得两条正面和两条负面的小贴士，帮助您了解当天的运势和建议。

## 常见问题

### 签到奖励什么时候发放？

签到奖励会立即发放到您的账户。经验值会自动添加到您的等级经验中，积分会添加到您的钱包。

### 我可以每天都签到吗？

是的，每天都可以签到一次。每日签到状态会在每天 UTC 0 点重置。

### 忘记签到可以补签吗？

补签功能正在开发中，目前客户端暂时不支持补签。
