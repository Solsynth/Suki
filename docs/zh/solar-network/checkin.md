# 签到系统

每日签到可以获得奖励！

## 如何签到

1. 进入 **仪表盘**
2. 点击签到按钮

## 签到奖励

| 奖励 | 说明 |
|------|------|
| 经验值 | 固定 100 点 |
| 积分 | 固定 10 积分 |

## 运势结果

签到时会随机获得一种运势：

| 运势 | 概率 |
|------|------|
| 下下签 | 10% |
| 下签 | 20% |
| 中签 | 40% |
| 上签 | 20% |
| 上上签 | 10% |

!!! note

    如果您在生日当天签到，将获得「特别签」特殊运势！

## 运势报告（第二版）

如果您的客户端使用签到 API 第二版或更新版本，系统会使用 AI（咩酱）生成详细的**运势报告**。包括：

- **运势提示**：两条正面和两条负面个性化提示
- **签诗**：主题运势诗
- **总评**：运势总评
- **详细建议**：详细的每日建议
- **愿望**：愿望方面的指引
- **爱情**：恋爱方面的指引
- **学业**：学业方面的指引
- **事业**：事业方面的指引
- **健康**：健康方面的指引
- **失物**：失物寻找方向
- **幸运色**：你的幸运颜色
- **幸运方位**：你的幸运方向
- **幸运时段**：你的幸运时段
- **幸运小物**：你的幸运物品
- **今日宜做**：今日推荐做的事
- **今日忌做**：今日该避免的事
- **小仪式**：每日小仪式建议

运势报告会根据你的个人资料、日历活动和特殊日子进行个性化生成。

## 签到小贴士

签到后会获得两条正面和两条负面的小贴士，帮助您了解当天的运势和建议。

## 常见问题

### 签到奖励什么时候发放？

签到奖励会立即发放到您的账户。经验值会自动添加到您的等级经验中，积分会添加到您的钱包。

### 我可以每天都签到吗？

是的，每天都可以签到一次。每日签到状态会在每天 UTC 0 点重置。

### 忘记签到可以补签吗？

目前补签功能暂不可用。
