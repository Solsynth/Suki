---
title: SOP 推送
description: Solar Network 的原生推送通知系统
---

SOP（Solar Network Push）是平台原生的通知推送系统。与第三方推送服务（APNs、FCM）不同，SOP 不需要外部推送令牌——它直接与你的认证会话配合工作。

## SOP 工作原理

1. **认证** 使用你的常规 Solar Network 会话
2. **注册** SOP 到你的设备/会话
3. **获取** 通知历史进行初始同步
4. **通过** SSE（服务器推送事件）实时流式推送通知
5. **重连** 断开后重新同步

## 为什么使用 SOP

- 不需要第三方推送令牌
- 任何支持 HTTP 客户端的平台都能使用
- 账户范围的通知列表作为真实来源
- 通过 SSE 流实时推送

## 关键行为

- SOP 具有设备感知去重功能——如果 SOP 处于活动状态，其他实时路径可能不会重复推送事件
- 列表 API 是遗漏通知的真实来源
- SSE 是实时更新通道
- 断开后始终重新连接并重新同步
