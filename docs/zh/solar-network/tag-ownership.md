---
title: 标签所有权
description: 了解标签如何被发布者拥有和保护
---

Solar Network 上的标签可以被发布者拥有，让他们能够控制标签的使用方式。

## 标签类型

| 类型 | 说明 |
|------|------|
| **未认领** | 所有人可用 |
| **已拥有** | 由发布者认领；由拥有者控制 |
| **受保护** | 只有拥有者可以在新帖子上使用该标签 |
| **活动** | 有时间限制的标签，到期后过期 |

## 认领标签

任何发布者都可以认领未被拥有的标签。认领后：

- 标签与你的发布者关联
- 你可以选择保护它
- 管理员可以将所有权分配给任何发布者

## 受保护标签

当标签被保护时：

- 只有拥有者可以在新帖子上使用该标签
- 已有帖子中的标签不受影响
- 受保护标签配额取决于你的特权等级：**3 + 3 × perkLevel**

## 活动标签

活动标签有时间限制：

- 在设定日期（`event_ends_at`）过期
- 过期后，新帖子不能再使用该标签
- 已有帖子永久保留该标签
