# 健身中心

追踪您的运动、记录身体数据、设定健身目标。

## 简介

健身中心是 Solar Network 的健身追踪服务，帮助您：

- 记录每次锻炼
- 追踪身体指标（体重、步数等）
- 设定并完成健身目标
- 与朋友比较运动数据

---

## 隐私设置

所有健身记录（锻炼、指标、目标）都有一个可见性设置：

| 设置 | 说明 |
|------|------|
| 私密 | 只有您自己可以看到 |
| 公开 | 嵌入到帖子后任何人都可以看到 |

默认情况下，所有记录都是私密的。如果您想让其他人看到，需要将可见性设置为「公开」。

---

## 记录锻炼

### 锻炼类型

| 类型 | 值 | 说明 |
|------|-----|------|
| 力量训练 | 0 | 举重、阻力训练 |
| 有氧运动 | 1 | 跑步、骑行、游泳 |
| HIIT | 2 | 高强度间歇训练 |
| 瑜伽 | 3 | 瑜伽、拉伸 |
| 其他 | 4 | 其他类型的运动 |

### 记录一次锻炼

您可以记录以下信息：

| 字段 | 说明 |
|------|------|
| 名称 | 锻炼的名称，如「晨跑」 |
| 类型 | 锻炼类型 |
| 开始时间 | 锻炼开始的时间 |
| 结束时间 | 锻炼结束的时间 |
| 时长 | 锻炼持续时间（如果未提供，会自动计算） |
| 消耗卡路里 | 估计消耗的卡路里 |
| 描述 | 锻炼的简要描述 |
| 备注 | 额外备注 |
| 可见性 | 私密/公开 |

### 附加数据（Meta）

除了基本信息，您还可以记录：

- **距离**：跑步、骑行等运动的距离
- **平均速度**：运动期间的均速
- **最大速度**：运动期间的最高速度
- **平均心率**：运动期间的平均心率
- **最大心率**：运动期间的最高心率
- **爬升高度**：累计爬升高度

### 在帖子中分享锻炼

当您将锻炼记录设置为「公开」后，可以在帖子中嵌入它：

```
在帖子中使用 fitness_reference 字段：
workout:{锻炼ID}
```

发帖时，帖子编辑器下方会提供一个选项，可以附加健身数据（锻炼、指标或目标）。

**数据同步推荐方式**

**iOS 设备**：推荐使用 **HealthKit** 同步您的运动数据。只需在设置中授权 Solar Network 访问健康数据，即可自动同步您的锻炼、步数、心率等数据。

**安卓设备**：由于 Google Fit API 已被 Google Health Connect 取代，目前暂不支持安卓设备的数据同步。我们正在努力在后续版本中添加支持。

## 记录身体指标

### 指标类型

| 类型 | 值 | 说明 |
|------|-----|------|
| 体重 | 0 | 体重（公斤/磅） |
| 体脂率 | 1 | 身体脂肪百分比 |
| 步数 | 2 | 每日步数 |
| 距离 | 3 | 行走/跑步距离 |
| 心率 | 4 | 静息或运动心率 |
| 睡眠 | 5 | 睡眠时长/质量 |
| 自定义 | 6 | 其他自定义指标 |

### 如何记录

记录您的身体指标，包括：
- 指标类型
- 数值
- 单位
- 记录时间
- 备注

系统会自动显示每种类型的最新记录，方便您追踪变化。

## 设定健身目标

### 目标类型

| 类型 | 值 | 说明 |
|------|-----|------|
| 减重 | 0 | 体重减轻目标 |
| 增肌 | 1 | 肌肉增加目标 |
| 耐力 | 2 | 跑步、骑行等耐力目标 |
| 步数 | 3 | 每日步数目标 |
| 自定义 | 4 | 其他自定义目标 |

### 目标状态

| 状态 | 值 | 说明 |
|------|-----|------|
| 进行中 | 0 | 目标正在进行中 |
| 已完成 | 1 | 目标已完成 |
| 已暂停 | 2 | 目标暂时暂停 |
| 已取消 | 3 | 目标已取消 |

### 自动进度更新

您可以绑定目标与特定的锻炼类型或指标类型：

- **绑定锻炼类型**：当记录指定类型的锻炼时，自动更新目标进度
- **绑定指标类型**：当记录指定类型的指标时，自动更新目标进度

例如：
- 绑定「减重」目标与「体重」指标
- 绑定「耐力」目标与「有氧运动」类型

### 重复目标

目标可以设置为重复：

| 重复类型 | 值 | 说明 |
|----------|-----|------|
| 无 | 0 | 不重复 |
| 每日 | 1 | 每天重复 |
| 每周 | 2 | 每周重复 |
| 每月 | 3 | 每月重复 |

您可以设置重复间隔和总重复次数。

### 目标完成

当目标状态变为「已完成」时：
- 如果是可重复目标，系统会自动创建下一个周期的目标
- 您会收到通知

## 常见问题

### 我可以导入其他应用的数据吗？

可以，系统支持通过外部 ID 进行批量导入。您可以使用 Strava 等应用的数据。

### 目标会自动更新进度吗？

是的，如果您绑定了锻炼类型或指标类型，系统会自动根据您的记录更新进度。您也可以手动更新进度。

### 锻炼记录可以删除吗？

可以，您可以删除任何锻炼、指标或目标记录。删除后相关数据将无法恢复。

### 我的数据安全吗？

是的，所有私密数据只有您自己可以查看。您可以随时将记录设置为公开或私密。

### 如何同步我的运动数据？

**iOS 用户**：推荐使用 **HealthKit**。在设置中授权 Solar Network 访问健康数据后，您的锻炼、步数、心率等数据会自动同步。

**安卓用户**：目前暂不支持 Google Fit/Health Connect 同步。我们正在努力在后续版本中添加支持。
