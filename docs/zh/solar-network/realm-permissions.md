---
title: 领域权限
description: 管理领域成员的访问控制
---

领域权限系统为领域成员提供细粒度的访问控制。作为领域所有者，你可以配置不同角色和个别用户可以执行的操作。

## 角色

每个领域有三个默认角色：

| 角色 | 等级 | 说明 |
|------|------|------|
| 所有者 | 100 | 对领域拥有完全控制权 |
| 管理员 | 50 | 可以管理内容和成员 |
| 普通 | 0 | 新加入者的默认角色 |

## 可用权限

| 权限 | 普通 | 管理员 | 所有者 |
|------|------|--------|--------|
| 聊天（发送消息） | ✅ | ✅ | ✅ |
| 发帖 | ✅ | ✅ | ✅ |
| 评论（回复帖子） | ✅ | ✅ | ✅ |
| 上传媒体 | ✅ | ✅ | ✅ |
| 管理帖子 | ❌ | ✅ | ✅ |
| 管理聊天（删除消息、禁言用户） | ❌ | ✅ | ✅ |
| 管理成员（邀请、移除、更改角色） | ❌ | ✅ | ✅ |
| 管理领域（编辑信息、标签、设置） | ❌ | ❌ | ✅ |

## 用户特定覆盖

除了基于角色的权限外，你还可以为个别用户设置权限覆盖。这些覆盖优先于用户角色权限。

用户权限值为 `null` 表示"回退到基于角色的权限"。

## 管理权限

### 查看/更新角色权限

前往领域设置查看和修改每个角色可以执行的操作。

### 设置用户覆盖

在成员列表中，你可以覆盖特定成员的权限——无论其角色如何，授予额外能力或限制特定操作。

## 帖子管理

管理员和所有者可以从领域中移除帖子。每次管理操作都会创建一条日志记录，包含：

- 管理员账户 ID
- 移除原因
- 时间戳

你可以在领域设置中查看管理日志。
