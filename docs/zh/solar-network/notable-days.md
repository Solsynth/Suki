---
title: 特殊日历
description: Solar Network 上的节日、活动和特殊日期
---

特殊日历系统用于追踪节日、活动、庆典和特殊场合。它为日历视图提供支持，并为每日签到运势提供背景参考。

## 日历类型

| 标签 | 说明 |
|------|------|
| 节日 | 官方节假日和公共假期 |
| 活动 | 特殊活动和场合 |
| 周年纪念 | 重要的周年纪念日 |
| 纪念日 | 缅怀纪念日 |
| 节庆 | 文化和季节性节庆 |

## 功能

- **多日周期：** 某些节日跨越多天（例如春节、劳动节）
- **地区支持：** 节日按地区划分（默认中国）
- **农历支持：** 支持农历事件
- **重复事件：** 支持阳历或农历的年度重复
- **本地化：** 名称和描述支持多语言

## 对你的影响

- **每日签到：** 特殊日历会影响你的运势报告
- **日历视图：** 在活动日历中查看即将到来的事件
- **社区：** 了解特殊场合的时间
