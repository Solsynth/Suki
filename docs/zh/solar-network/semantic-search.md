语义搜索允许你使用自然语言查询来查找帖子，超越简单的关键词匹配。

## 工作原理

帖子使用 AI 生成的嵌入向量（向量表示）进行索引。搜索时：

1. 你的查询被转换为嵌入向量
2. 系统使用余弦距离找到相似的帖子
3. 结果按语义相关性排序

## 搜索模式

| 模式 | 说明 |
|------|------|
| 语义（默认） | 基于向量的相似度搜索 |
| 全文搜索 | 传统关键词搜索 |

你可以使用 `searchEngine` 查询参数切换模式：

```
GET /api/posts?search=your+query
GET /api/posts?search=your+query&searchEngine=fulltext
```

## 何时使用哪种模式

- **语义搜索：** 最适合概念性查询、查找相关内容、自然语言问题
- **全文搜索：** 最适合精确关键词匹配、特定术语、用户名
