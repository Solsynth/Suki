---
title: 源点钱包
---

源能点数（源点）是 Solar Network 上的虚拟币种，
其可以用来支付 Solar Network 上的一些开销。

## 钱包

您可以在登录之后在 Solian 中转到「钱包」页面来创建一个源点钱包，
在这之前您不会收到任何被动源点收入。

同时您也可以在「钱包」页面查看您当前的余额和交易详情。

### 多钱包支持

Solar Network 现已支持每个账户创建**多个个人钱包**。你可以创建不同用途的钱包（例如日常消费钱包、储蓄钱包等）。

- **个人钱包配额**: 每个账户最多可创建 **3 个个人钱包**。
- **默认钱包**: 其中一个钱包被标记为默认钱包，用于向后兼容的流程。
- **领域钱包**: 可以为领域（Realm）创建钱包供有权限的成员共同使用。

每个钱包可以选填**公开钱包 ID**（例如 `DNW-ABCD-EFGH-JKLM`），方便转账。

## 支付

在支付之前，为了确保是您本人使用的源点钱包来支付，您需要在「帐号设置」中创建一个 PIN Code 验证因素，其的可信度为 `0`，无法用来登录。只会用来验证身份。

Solian 的客户端在第一次输入 PIN Code 之后会在本地安全的情况下存储加密的 PIN Code，以便提供生物识别快速验证。

源点（Source Points aka NSP）是 Solar Network 的基本货币单位，你可以用来购买一些虚拟资源。

**注意：Solar Network 的源点不支持储值，任何售卖源点的都是诈骗。**

## 获得

你可以通过每日签到来获得源点。不过在用户创建钱包之前，签到不会奖励任何源点。

同时，你也可以通过积极的在 Solar Network 上与活动来赚取源点，在达到一定等级之后，你可以加入「创作奖励计划」来在发布帖子、贴图、以及其他内容的时候获得源点奖励。

> **你知道吗？**
> 由 Solsynth 运营的 GoatCraft Minecraft 服务器内的货币与源点是互通的，虽然属于 Solar Network 的范围，但是也不失为一种赚取源点的好方式。

## 消费

在 Solar Network 上需要源点的消费有很多，下面是一部分：

- 问 SN/咩酱问题
- 奖励帖子
- 兑换恒星计划

- 上传更大的文件
- 购买彩票

## 金太阳点

金点是 Solar Network 上比较稀有的货币。当前的唯一获得方法是给<del>羊打钱</del>购买恒星计划才会得到。

!!! note

    在应用内使用源点兑换的恒星计划不会得到金点奖励。

根据订阅的恒星计划等级不同，得到的金点数量也不同。目前金点的主要用途是给领域提供 Boost，以获得更多功能。
我们正在努力扩充金点的能力。敬请期待！
