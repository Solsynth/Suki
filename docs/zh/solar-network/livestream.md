直播功能让你能够使用 LiveKit 作为实时基础设施创建和广播直播视频流。

## 功能概览

- 通过 OBS 进行 RTMP 流媒体传输
- 观众通过 Web 端或移动端观看
- 支持 HLS 播放，兼容更多设备
- 将直播推送到 YouTube、Bilibili 等外部平台
- 实时聊天与观众互动
- 观众打赏支持

## 开始直播

### 使用 OBS（电脑直播）

1. **创建直播**
   在直播页面点击「创建直播」，填写标题、描述等信息。

2. **获取推流地址**
   点击「开始直播」，获取 RTMP 地址和推流密钥。

3. **配置 OBS**
   - 服务：自定义
   - 服务器：填写获取的 RTMP 地址
   - 推流密钥：填写获取的推流密钥

4. **开始推流**
   在 OBS 中点击「开始推流」即可开始直播。

### 手机/网页直播

1. 创建直播后，选择「应用内直播」模式
2. 通过直播间获取观看链接
3. 直接在应用中开播

## 观众观看

1. 在直播列表浏览正在直播的内容
2. 点击进入直播间
3. 登录后可参与聊天和打赏

## 直播聊天

- 所有观众都可以发送聊天消息
- 主播可以删除不当消息或禁言用户
- 消息会实时推送给所有观看者

## 直播打赏

观众可以通过打赏来支持主播：

- 打赏会增加直播在发现页的排名
- 打赏可附带消息
- 主播可高亮展示打赏（1 积分 = 2 秒高亮时间）

## 推送到外部平台

直播时可以同时推送到其他平台：

1. 开始直播后，点击「推送到外部平台」
2. 填写目标平台的 RTMP 地址
3. 直播会同时在多个平台播出

## 常见问题

**直播显示未开始？**
- 确认 OBS 已连接并正在推流
- 检查直播状态是否为「直播中」

**无法更新直播信息？**
- 直播进行中无法修改信息
- 需要先结束直播再修改

**打赏后排名没有变化？**
- 排名计算有延迟，稍等片刻后刷新
