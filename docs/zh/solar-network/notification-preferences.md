---
title: 通知偏好
---

控制您接收各类通知的方式，你可以在帐号设置中调整这些设定。

## 简介

您可以为不同的通知类型设置接收方式：

- **正常** - 完全接收（存储 + 推送 + 实时通知）
- **静默** - 仅存储（不推送，不实时通知）
- **拒收** - 完全关闭（不存储，不发送）

## 三种接收级别

| 级别 | 说明 |
|------|------|
| 正常 | 您会收到推送通知、实时通知，也会保存在通知记录中 |
| 静默 | 通知只保存到通知中心，但不会打扰您（无推送无提醒） |
| 拒收 | 完全不接收此类通知，也不会保存 |

## 常见通知类型

| 通知类型 | 说明 |
|----------|------|
| `posts.mentions.new` | 帖子中有人提及您 |
| `post.replies` | 帖子收到回复 |
| `posts.reactions.new` | 帖子收到反应（点赞等） |
| `posts.awards.new` | 帖子获得徽章 |
| `subscriptions.discontinued_in_app` | 订阅已取消 |
| `subscriptions.begun` | 订阅已开启 |
| `gifts.claimed` | 礼物被领取 |
| `wallets.transactions` | 钱包交易记录 |
| `auth.verification` | 身份验证 |
| `invites.realms` | 领域邀请 |
| `livestream.started` | 直播开始 |

## 如何设置

您可以在设置中管理这些通知偏好：

1. 进入**帐号** > **帐号设置** > **Notification Preferences (或 通知偏好)**
2. 找到您想调整的通知类型
3. 选择接收级别：**正常**、**静默**或**拒收**

例如：

- 如果您不想错过任何重要消息，保持**正常**
- 如果您想查看但不想被打扰，选择**静默**
- 如果您完全不关心某类通知，选择**拒收**
