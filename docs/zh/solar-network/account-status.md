# 账号状态与在线提示

了解如何管理您的在线状态、个性化设置以及查看他人的状态。

## 在线状态类型

Solar Network 提供四种在线状态，您可以根据需求自由切换：

| 状态类型 | 值 | 说明 | 他人能看到什么 |
|----------|-----|------|----------------|
| 正常 | 0 | 正常可见状态 | 显示「在线」 |
| 忙碌 | 1 | 可见的忙碌状态 | 显示「忙碌」 |
| 请勿打扰 | 2 | 可见的免打扰状态 | 显示「请勿打扰」 |
| 隐身 | 3 | 完全隐藏在线状态 | 显示「离线」 |

### 状态说明

- **正常**：您的默认状态，其他人可以正常看到您在线
- **忙碌**：表示您正在忙其他事情，不希望被打扰
- **请勿打扰**：开启后，推送通知将被禁用
- **隐身**：完全隐藏您的在线状态，其他人会看到您离线

!!! tip

    如果您设置了「隐身」状态，他人查看您的个人资料时会显示「离线」，而不会知道您实际在线。

## 设置您的状态

### 设置位置

进入 **账号** > 单击 **状态**

### 可设置的项目

| 字段 | 说明 | 限制 |
|------|------|------|
| 状态类型 | 选择您的在线状态 | 必选 |
| 状态标签 | 自定义状态文字 | 最长 1024 字符 |
| 图标 | 状态图标 | 最多 128 字符 |
| 清除时间 | 自动清除状态的时间 | 可选 |

### 自动清除

您可以设置状态在特定时间自动清除。例如：

- 开会时设置为「忙碌」，设定 1 小时后自动恢复为「正常」
- 休息时设置为「请勿打扰」，设定下班时间自动恢复

## 状态态度

除了在线状态，您还可以设置「状态态度」来表示您当前的心情：

| 态度类型 | 说明 |
|----------|------|
| 正面 | 表示积极向上的心情 |
| 中立 | 表示一般的心情 |
| 负面 | 表示消极或低落的心情 |

状态态度会显示到您的活动日历中。

## 聊天室在线成员

在聊天室中，您可以查看当前在线的成员。

### 群组聊天室

在群组聊天室中，您可以看到：

- 在线人数
- 在线成员的昵称列表
- 在线成员的详细信息（点击可查看）

### 私信会话

在私信中，您可以看到：

- 对方的在线状态
- 对方的具体状态（如「正在开会」）
- 对方的在线/离线状态

## 在线状态常见问题

### 为什么我的状态显示不正确？

状态显示可能会有短暂的延迟，因为系统会缓存您的状态信息。如果您刚刚更改了状态，请稍等片刻再查看。

### 为什么别人看到我离线但我实际在线？

这可能是因为您设置了「隐身」状态。在这种情况下，您的在线状态会对其他用户隐藏。

或者客户端的 WebSocket 链接断开了，没有准确反馈在线状态到服务器。

### 我可以同时设置多个状态吗？

不可以。您只能设置一个当前状态。但您可以设置「清除时间」，让状态在指定时间后自动恢复为默认。
