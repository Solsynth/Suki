# 成就与任务系统

通过完成任务和成就，获得经验值、积分和徽章！

## 简介

Solar Network 有一个完整的成就和任务系统。当您进行特定操作（如发布帖子、发送消息、加入领域等）时，系统会自动记录您的进度。达成目标后，您将自动获得奖励。

### 奖励类型

| 奖励类型 | 说明 |
|----------|------|
| 经验值 | 提升您的账户等级 |
| 积分 | 可在钱包中使用 |
| 徽章 | 展示在您的个人主页上 |

### 进度类型

- **普通进度**：每完成一次操作累加一次
- **连续进度**：需要连续多天保持活动（断签会重置）

## 如何查看进度

1. 进入您的个人主页
2. 点击「成就」或「任务」标签
3. 查看您的进度和已完成的项目

!!! tip

    连续成就（如连续发布、连续登录）需要您保持每日坚持，中断一天就会重置计数哦！
