# 处罚系统

了解 Solar Network 的处罚机制以及如何避免违规。

## 什么是处罚

处罚是 Solar Network 对违反用户协议或社区规则的用户采取的管理措施。处罚的目的是维护社区秩序，保护其他用户的权益。

## 处罚类型

| 处罚类型 | 说明 |
|----------|------|
| 警告 | 提醒您注意自己的行为，累计过多可能升级 |
| 限制登录 | 暂时无法登录账号 |
| 禁用账号 | 账号被永久禁用 |
| 权限限制 | 禁止使用特定功能 |

### 权限限制

当您收到权限限制处罚时，某些功能将被暂时禁用，例如：

- 无法发送聊天消息
- 无法上传文件
- 无法使用钱包功能

权限限制可以是单项的（如仅限制聊天），也可以是多项的（限制多种功能）。

## 处罚与信用分的关联

Solar Network 使用**社会信用分**系统来评估用户的可信度。处罚会影响您的信用分：

- **警告**：轻微扣分
- **权限限制**：根据违规程度扣分
- **限制登录**：较大幅度扣分
- **禁用账号**：大幅扣分，可能导致账号信用冻结

信用分越低，您在平台上的活动会受到越多限制。反之，保持良好的信用分可以让您享受更多功能和服务。

详细了解：[社会信用系统](./social-credit.md)

## 处罚通知

当您收到处罚时，系统会：

1. **推送通知**：向您发送应用内通知，说明处罚原因和持续时间
2. **解除通知**：当处罚到期解除时，您也会收到通知

!!! tip

    请确保您的通知设置已开启，以便及时了解账号状态。

## 常见处罚原因

- 发布违规内容
- 骚扰其他用户
- 垃圾信息或刷屏
- 违规使用自动化工具
- 尝试规避平台规则
- 信用分过低导致功能受限

## 常见问题

### 我的处罚会持续多久？

处罚的持续时间取决于违规的严重程度和历史记录。轻微违规可能只会收到警告，而严重违规可能会导致更长时间的限制。

### 我可以申诉吗？

如果您认为处罚有误，可以通过客服渠道提交申诉。我们会重新审核您的案例。

### 处罚到期后会自动解除吗？

是的，大多数处罚都会在到期后自动解除，您的权限将恢复正常。系统会发送通知告知您处罚已解除。

### 警告会累积吗？

是的，多次警告可能会升级为更严厉的处罚。请务必遵守社区规则，避免重复违规。

### 处罚会影响我的信用分吗？

会的。每次处罚都会影响您的社会信用分。信用分降低可能导致：
- 部分功能受限
- 在推荐和动态中曝光降低
- 更多限制

## 如何避免处罚

1. **遵守用户协议**：了解并遵守 Solar Network 的用户协议
2. **尊重他人**：友善对待其他用户，不进行骚扰
3. **不发布违规内容**：避免发布违法、暴力或不当内容
4. **不使用自动化工具**：未经允许不使用机器人或自动化脚本
5. **保护账号安全**：不要将账号借给他人，避免被盗用
6. **保持良好信用**：积极参与社区互动，维护良好的信用分
