---
title: 屏蔽与静音
description: 管理你与其他用户的互动
---

Solar Network 提供两种机制来控制你与其他用户的互动：**屏蔽**和**静音**。

## 屏蔽

屏蔽用户会产生**双向**互动限制。你和被屏蔽的用户都无法看到或互动对方的内容。

当你屏蔽某人时：

- 他们的帖子会从你的动态和时间线中隐藏
- 你无法回复、反应、转发、助推或奖励他们的帖子
- 他们的消息和通知被屏蔽
- 你们之间的私信被阻止
- 他们从搜索和自动补全中隐藏
- 你们之间的关注请求被阻止

## 静音

静音用户是**单向的**。你停止看到被静音用户的内容，但他们不受影响。

当你静音某人时：

- 他们的帖子会从你的动态和时间线中隐藏
- 你停止接收他们的通知
- 他们仍然可以正常与你的内容互动

## 主要区别

| | 屏蔽 | 静音 |
|---|------|------|
| 方向 | 双向 | 单向 |
| 他们看到你的内容 | 否 | 是 |
| 你看到他们的内容 | 否 | 否 |
| 私信 | 被阻止 | 允许 |
| 通知 | 被屏蔽 | 被屏蔽 |

## 管理屏蔽/静音列表

你可以在账户设置中查看和管理已屏蔽和已静音的用户。你可以随时解除屏蔽或取消静音。
