# 身份验证方式

了解 Solar Network 提供的多种身份验证方式，保护您的账号安全。

## 什么是身份验证方式

身份验证方式（也称为「双因素认证」或「两步验证」）是为您的账号添加的额外安全层。即使密码被他人获取，没有您的第二步验证，他们也无法登录您的账号。

!!! warning

    **重要提示**：在添加任何其他身份验证方式之前，**必须先设置恢复码**。这是您账号的最终恢复手段。

## 可用的身份验证方式

### 密码

您注册时设置的账号密码。这是登录的基础验证方式。

### 电子邮件验证码

发送到您注册邮箱的一次性验证码。每次登录时都会发送新的验证码。

### 时序验证码 (TOTP)

由 TOTP 验证器（如 Google Authenticator、Authy 等）生成的一次性验证码。这种方式不需要网络连接，更加安全可靠。

### 应用内通知

通过 Solar Network 应用内推送通知发送的一次性验证码。无需手动输入，方便快捷。

### PIN 码

6 位数字的 PIN 码。**注意：PIN 码不能用于登录**，但在进行高风险操作（如删除账号、大额转账等）时，系统会要求输入 PIN 码进行确认。

### 恢复码

用于账号恢复的备用代码。当您无法使用其他验证方式（如更换手机、丢失邮箱访问权限等）时，恢复码可以帮助您找回账号。

!!! warning

    恢复码只能显示一次，请务必妥善保存！建议使用密码管理器存储。

### 实体通行证

你可以从我们的商店 / 参加活动以获得实体通行证的周边。用其来登陆。

### 通行密钥

使用设备的生物识别功能（指纹、Face ID）或设备密码进行验证。这是最现代、最便捷的验证方式，只需指纹或面部识别即可登录。

---

## 验证方式设置顺序

强烈建议按以下顺序设置：

1. **首先**：设置恢复码（必须）
2. **其次**：设置通行密钥或 TOTP（推荐）
3. **可选**：添加电子邮件验证码、应用内通知等作为备用

---

## 管理您的验证方式

### 设置位置

进入 **账号** > **账号设置** > **验证因子**

### 可执行的操作

| 操作 | 说明 |
|------|------|
| 查看 | 查看已添加的验证方式及状态 |
| 添加 | 添加新的验证方式 |
| 启用 | 激活已添加但未启用的验证方式 |
| 禁用 | 暂时关闭某个验证方式（可重新启用） |
| 删除 | 永久删除某个验证方式 |

### 启用验证方式

某些验证方式在添加后需要启用才能使用。启用时需要输入该方式生成的验证码进行确认。

### 删除验证方式

删除验证方式后，该方式将无法再用于登录。删除后可以重新添加。

---

## 信任等级

不同的验证方式有不同的信任等级：

| 验证方式 | 信任等级 | 说明 |
|----------|----------|------|
| 恢复码 | 0（最低） | 仅用于账号恢复 |
| 密码 | 1 | 基础登录方式 |
| PIN 码 | 1 | 用于高风险操作确认 |
| 电子邮件验证码 | 2 | 通过邮件发送 |
| 应用内通知 | 2 | 通过应用推送 |
| 时序验证码 (TOTP) | 3（最高） | 最安全的验证方式 |

登录时，系统会根据您设置的验证方式，要求通过相应数量的验证步骤。

## 常见问题

### 我可以添加多个同类型的验证方式吗？

不可以。每种类型的验证方式只能添加一个。

### 如果我丢失了手机，无法接收验证码怎么办？

如果您已经设置了恢复码，可以使用恢复码登录并重新设置其他验证方式。

### 我可以只用一个验证方式登录吗？

可以，但建议至少设置两种验证方式以确保账号安全。密码是必须的，加上其他任何一种验证方式即可开启双因素认证。

### 删除某个验证方式会影响登录吗？

不会，但您将失去该验证方式提供的安全保障。建议始终保持至少两种有效的验证方式。

### 什么是高风险操作？

高风险操作包括：

- 删除账号
- 大额转账
- 修改安全设置
- 删除重要数据

这些操作可能需要您输入 PIN 码进行确认。

---

## 安全建议

1. **始终先设置恢复码**：这是您账号的「救命稻草」
2. **至少设置两种验证方式**：推荐密码 + 通行密钥/TOTP
3. **妥善保管恢复码**：建议使用密码管理器
4. **定期检查验证方式**：确保所有验证方式都可用
5. **不要将验证码告诉他人**：Solar Network 员工不会向您索取验证码
