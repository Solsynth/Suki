---
title: 问卷
description: 创建问卷并收集社区回复
---

问卷允许发布者创建调查问卷来收集社区回复。

## 问卷生命周期

| 状态 | 说明 |
|------|------|
| 草稿 | 可编辑；不接受回复 |
| 已发布 | 不可编辑；在 `ended_at` 之前接受回复 |
| 已归档 | 完全锁定；不可编辑，不接受新回复 |

要修改已发布的问卷，请将其克隆为新的草稿。已有的回答仍绑定到已冻结的发布版本。

## 问题类型

| 类型 | 说明 |
|------|------|
| 单选 | 选择一个选项 |
| 多选 | 选择多个选项 |
| 是/否 | 二选一 |
| 评分 | 数字评分 |
| 自由文本 | 开放式文本回答 |

## 创建问卷

1. 前往创作者中心
2. 创建新问卷并设置问题
3. 设置结束日期（可选）
4. 准备就绪后发布

## 回复

- 任何人都可以回答已发布的问卷
- 回复可以是匿名的或实名的
- 问卷结果对发布者可见
- 非匿名回复对其他受访者可见

## 嵌入

问卷可以嵌入到帖子和聊天消息中，让受众在看到内容的地方直接参与。
