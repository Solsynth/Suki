---
title: 聊天表情反应
description: 用表情反应表达对聊天消息的态度
---

聊天表情反应让你为聊天消息添加表情符号式的反应。

## 可用反应

| 符号 | 说明 |
|------|------|
| `thumb_up` | 点赞 |
| `thumb_down` | 踩 |
| `just_okay` | 一般般 |
| `cry` | 哭泣 |
| `confuse` | 困惑 |
| `clap` | 鼓掌 |
| `laugh` | 大笑 |
| `angry` | 生气 |
| `party` | 庆祝 |
| `pray` | 祈祷 |
| `heart` | 爱心 |

自定义表情反应符号需要活跃的订阅。

## 反应机制

- 为任何消息开启或关闭反应
- 每个消息上显示反应总数
- 所有房间成员通过 WebSocket 实时查看反应
- 反应持久保存，任何有房间访问权限的人都能看到
