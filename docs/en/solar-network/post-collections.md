Post Collections allow publishers to create curated, ordered lists of posts with built-in navigation.

## Overview

Collections are publisher-scoped resources that let you group related posts together. Readers can navigate through a collection using previous/next links.

## Creating a Collection

When creating a collection, you can set:

- **Name:** Display name for the collection
- **Description:** Brief description of what the collection contains
- **Icon:** Custom icon image
- **Background:** Background image for the collection page

## Managing Posts

- Add posts to a collection in any order you choose
- Posts can appear in collections owned by other publishers
- Collection management requires **Editor** role or higher on the publisher

## Reader Experience

When viewing a post that belongs to a collection, readers see:

- Which collections the post belongs to
- Previous/next navigation within each collection
- The full collection page with all posts in order

## Visibility

Collections respect post visibility settings. If a reader doesn't have access to a post (e.g., it's friends-only), that post is hidden from the collection view.
