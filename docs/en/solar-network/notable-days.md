Notable Days is a system that tracks holidays, events, festivals, and special occasions. It powers the calendar view and provides context for daily check-in fortunes.

## Types of Notable Days

| Tag | Description |
|-----|-------------|
| Holiday | Official holidays and public breaks |
| Event | Special events and occasions |
| Anniversary | Important anniversaries |
| Memorial | Days of remembrance |
| Festival | Cultural and seasonal festivals |

## Features

- **Multi-day periods:** Some holidays span multiple days (e.g., Spring Festival, Labour Day)
- **Regional support:** Holidays are region-specific (defaults to China)
- **Lunar calendar:** Supports lunar calendar events
- **Recurring events:** Annual recurrence with solar or lunar patterns
- **Localization:** Names and descriptions support multiple languages

## How It Affects You

- **Daily check-in:** Notable days influence your fortune report
- **Calendar view:** See upcoming events in your activity calendar
- **Community:** Know when special occasions are happening
