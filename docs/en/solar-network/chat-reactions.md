---
title: Chat Reactions
description: Express yourself with reactions on chat messages
---

Chat Reactions let you add emoji-style reactions to messages in chat rooms.

## Available Reactions

| Symbol | Description |
|--------|-------------|
| `thumb_up` | Thumbs up |
| `thumb_down` | Thumbs down |
| `just_okay` | Just okay |
| `cry` | Crying face |
| `confuse` | Confused face |
| `clap` | Clapping hands |
| `laugh` | Laughing face |
| `angry` | Angry face |
| `party` | Party/celebration |
| `pray` | Praying hands |
| `heart` | Heart/like |

Custom reaction symbols require an active subscription.

## How Reactions Work

- Toggle a reaction on or off for any message
- Reaction totals are displayed on each message
- All room members see reactions in real-time via WebSocket
- Reactions persist and are visible to anyone with access to the room
