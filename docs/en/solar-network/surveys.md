Surveys let publishers create questionnaires to collect responses from the community.

## Survey Lifecycle

| Status | Description |
|--------|-------------|
| Draft | Editable; no responses accepted |
| Published | Immutable; responses accepted until `ended_at` |
| Archived | Fully locked; no edits, no new responses |

To revise a published survey, clone it to a new draft. Existing answers stay bound to the frozen published version.

## Question Types

| Type | Description |
|------|-------------|
| Single Choice | Pick one option |
| Multiple Choice | Pick multiple options |
| Yes/No | Binary choice |
| Rating | Numeric rating scale |
| Free Text | Open-ended text response |

## Creating a Survey

1. Go to your Creator Center
2. Create a new survey with your questions
3. Set the end date (optional)
4. Publish when ready

## Responses

- Anyone can answer published surveys
- Responses can be anonymous or attributed
- Survey results are visible to the publisher
- Non-anonymous responses are visible to other respondents

## Embedding

Surveys can be embedded in posts and chat messages, allowing your audience to participate directly where they see the content.
