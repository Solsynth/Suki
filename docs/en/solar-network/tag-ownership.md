---
title: Tag Ownership
description: Understand how tags can be owned and protected by publishers
---

Tags on Solar Network can be owned by publishers, giving them control over how their tags are used.

## Tag Types

| Type | Description |
|------|-------------|
| **Unclaimed** | Available to everyone |
| **Owned** | Claimed by a publisher; controlled by the owner |
| **Protected** | Only the owning publisher can use this tag on new posts |
| **Event** | Time-limited tag that expires after a set date |

## Claiming a Tag

Any publisher can claim an unowned tag. Once claimed:

- The tag is associated with your publisher
- You can choose to protect it
- Admins can assign ownership to any publisher

## Protected Tags

When a tag is protected:

- Only the owning publisher can use it on new posts
- Existing posts with the tag are unaffected
- Your protected tag quota depends on your perk level: **3 + 3 × perkLevel**

## Event Tags

Event tags are time-limited:

- They expire at a set date (`event_ends_at`)
- After expiration, new posts cannot use the tag
- Existing posts keep the tag indefinitely
